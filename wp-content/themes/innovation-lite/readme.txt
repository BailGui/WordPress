Innovation Lite Responsive and Professional Theme

== Description ==

Innovation is CSS3 Powered and WordPress Latest Version Ready Responsive Theme.  Innovation has Multi Layered Slider, Portfolio and Animated Staff Box Items.  Super Elegant and Professional Innovation Lite Theme will be adjusted automatically with all smart devices if the Responsive Layout be selected.  Full Wide Template can be used for showing showcase or other pages. Featured Boxes, Portfolio and Staff Boxes can represent your company to the world.

== Frequently Asked Questions ==

= What to do after Theme Installation ? =
After installing the Theme Please go to Appearance > Customize. You can set the Slide Images, Featured Boxes, Staffs, Portfolio Boxes, Social Links etc. there.


= Why is Innovation? =
Innovation is for Personal and Corporate Web Sites

= Can it be used for Personal or Company Websites? =
You can use for Personal to Corporate Sites

= Is it only for Blogs ? =
No, It is for Blogs, Company, Personal Websites etc.
